📋 Instrucciones para guardar cambios permanentemente
  1. Haz click en Generar código arriba
  2. Copia todo el contenido del área de texto
  3. Abre el archivo js/main.js en tu editor
  4. Reemplaza la sección const DEFAULT_INVENTORY = [...]; con el código copiado
  5. Guarda el archivo. ¡Listo!
